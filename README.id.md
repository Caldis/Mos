## Features

- Bikin scroll pakai mouse wheel sehalus pakai touchpad.
- Bisa atur gaya scroll sesuka hati — dari kecepatan sampai akselerasinya.
- Mau bedain cara scroll mouse dan touchpad? Bisa banget.
- Ada grafik kece yang nunjukin gimana scroll kamu bekerja.
- Punya fitur blokir/izin scroll per aplikasi, langsung dari dalem.
- Dibangun pakai Swift, jadi ringan dan responsif.


## Download & Install

### Homebrew

Kalau kamu mau instal aplikasi ini lewat [Homebrew](https://brew.sh):

```bash
$ brew install --cask mos
```

Aplikasi ada di `/Applications/Mos.app`.

Untuk Update Aplikasi

```bash
$ brew update
$ brew reinstall mos
```

Keluar dan jalankan lagi aplikasi
