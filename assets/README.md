# Assets

Repository-owned media that is not a generated website artifact lives here.

- `readme/`: README screenshots, app icon previews, and archived README images.
- `source/app/`: app visual source material and historical exports. These files
  are not loaded from the app bundle directly; runtime copies belong in
  `Mos/Assets.xcassets`.

Packaging-specific build inputs stay next to their packaging scripts, for
example `packaging/dmg/assets/`.
