# App Source Assets

Source files and legacy exports for app UI assets.

- `donate/`: source copies for donation logos and QR images. Runtime versions
  are imported into `Mos/Assets.xcassets/Donate/`.
- `icons/`: icon source material and SF Symbols-style exports. Runtime versions
  are imported into the relevant `Mos/Assets.xcassets` groups.
- `designs.sketch`: legacy Sketch design source.
