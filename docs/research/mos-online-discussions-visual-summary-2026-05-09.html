AI 友好化路线

仓库已经有 llms.txt 和 agent discovery；下一步重点是让 AI 能帮助用户排障、解释配置并生成安全草案。

1. Mos Support Bundle

导出 Markdown + JSON：版本、macOS、权限状态、输入监控状态、设备列表、Logi HID++ 能力、活跃 profile、异常 app。用户可直接贴给 AI 或 issue。

2. Machine-readable config schema

为滚动 profile、按钮绑定、Logi CID、快捷动作、open-target payload 提供 JSON Schema，支持 AI 生成、解释和安全校验。

3. Agent-facing docs matrix

把“Mos 适合/不适合什么”“常见冲突”“推荐设置”做成短、可引用、版本化 Markdown，并与 website llms.txt 同步。

4. Local automation surface

CLI / URL scheme / AppleScript / Shortcuts：status、export、validate、open preferences。写操作先 dry-run 和 UI confirmation。

5. Structured changelog

为 added/fixed/logi/accessibility/docs 等变更加 JSON/YAML sidecar，供 appcast、官网、AI 回答和回归筛选复用。

6. Read-only MCP last

MCP 放最后：本地 stdio、默认不常驻、只读优先，不直接修改输入行为或运行用户脚本。
